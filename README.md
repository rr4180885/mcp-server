# mcp-server

GitHub MCP server built with TypeScript, Express, and the Model Context Protocol SDK.

## Scripts

- `npm run dev` — HTTP MCP server
- `npm run dev:stdio` — stdio MCP server
- `npm run build` — TypeScript build
- `npm start` — run built server

## Setup

1. Create a `.env` with `GITHUB_TOKEN=...` (never commit this file)
2. `npm install`
3. `npm run dev`
